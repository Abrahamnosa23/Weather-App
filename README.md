## A weather app that you can use to check the weather of any city in the world.

- **Student Name:** Abraham Aigbokhan
- **Student ID:** ALT/SOE/BAR/026/0210

## Description

I built this simple Weather App using HTML, CSS and JavaScript. I used Open-Meteo Geocoding API to convert a city name into coordinates, and the Open-Meteo Forecast API to retrieve current weather and a 5-day forecast. My JavaScript code uses async/await for network requests, maps WMO weather codes to human-readable descriptions and simple icons, and updates the DOM to show city, temperature (°C), condition, humidity, wind speed (km/h), UV category (Low/Moderate/High) and a five-day forecast.

## How to run

1. Open `[Weather App](https://abrahamnosa23.github.io/Weather-App/)` in your browser.
2. Enter a city name and click Search.
